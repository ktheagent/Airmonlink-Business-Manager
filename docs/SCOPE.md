# Version 1 scope

This release is a complete offline MVP for a single Windows computer. It is not yet a multi-user accounting suite, payment processor, tax filing system, cloud service or pharmacy clinical system.

## Implemented

POS, stock control, product management, customer and supplier directories, customer credit sales, outstanding-balance payment recording, expenses, sales reporting, CSV/PDF exports, A4/Letter report printing, 57/57/80 mm sales receipts, printer test page, business profile and local backups.

## Planned after field testing

Staff roles, barcode label printing, historical receipt reprinting, returns, purchase orders, detailed account ledgers, audit logs, automatic backups, licensing, updates and optional multi-branch cloud synchronization.

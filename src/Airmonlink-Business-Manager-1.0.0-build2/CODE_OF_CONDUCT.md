# Contributor Covenant Code of Conduct

## Our pledge

We pledge to make participation in this project a respectful, harassment-free experience for everyone.

## Expected behaviour

Be respectful, focus criticism on the work, protect user data, avoid discriminatory or threatening language, and collaborate in good faith.

## Unacceptable behaviour

Harassment, personal attacks, publishing private information, deliberate disruption, malicious code, credential exposure and misuse of project communication channels are not accepted.

## Enforcement

Project maintainers may edit or remove contributions and may temporarily or permanently restrict participation when conduct threatens the community, users or project security. Security reports should follow `SECURITY.md` rather than public issue discussions.
